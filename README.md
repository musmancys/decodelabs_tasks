# 📖 Overview

This repository contains four cybersecurity-focused Python projects that demonstrate fundamental security concepts including password auditing, phishing detection, vulnerability assessment, and classical encryption.

These projects were created to strengthen practical Python programming skills while learning core cybersecurity principles.

---

# 🚀 Projects Included

## 🔐 Password Strength Checker

Analyzes passwords and determines their security level.

### Features
- Password strength analysis
- Checks for uppercase and lowercase characters
- Detects numbers and special symbols
- Identifies common passwords
- Provides security recommendations

---

## 🎣 Phishing Awareness Analysis

Analyzes emails or messages for common phishing indicators.

### Features
- Detects suspicious keywords
- Finds shortened URLs
- Detects generic greetings
- Highlights potential phishing techniques
- Generates a simple security assessment

---

## 🛡️ Vulnerability Checklist

An interactive security assessment tool.

### Features
- Password security checks
- Basic system security questions
- Identifies common vulnerabilities
- Generates personalized security recommendations

---

## 🔒 Basic Encryption & Decryption

Implements the Caesar Cipher algorithm.

### Features
- Encrypt text
- Decrypt encrypted messages
- Custom shift key
- Input validation
- Command-line interface

# ▶️ Running the Projects

Clone the repository

```bash
git clone https://github.com/musmancys/Decode-Labs.git
```

Navigate to the project

```bash
cd Decode-Labs
```

Run any Python file

```bash
python password_strength_checker.py
```

or

```bash
python Phishing_Analysis.py
```
